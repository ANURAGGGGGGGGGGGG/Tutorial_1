Status code node js

200 OK : The request was succesful, and the server has returned the requested data.
201 Created : The request was successful, and a new resource was created as a result.
204 No Content : The request was successful, but there is no data to return (often used for DELETE requests.)
400 Bad Request: The server cannot understand the request due to a client error (e.g., maformed request syntax).
401 Unauthorized: The client must authenticate itself to get the requested response.
403 Forbidden: The client does not have the neccessary permission to access the requested resorce.
404 Not Found: The requested resource is not found on the server.
422 Validation error: When you feild is not present
500 Internal Server Error: The server encountered an error while processing the request
